# Custom lexicons

Place your own hand-authored lexicon JSON files here.

## Folder conventions

- **`lexicons/custom/`** (this folder) — Your own hand-authored lexicons.
- **`lexicons/pulled/`** — Downloaded via `lex-cli pull` from the atproto registry.
  Generated from the `pull.sources` list in `lex.config.js`.
- **`lexicons/generated/`** — Auto-generated by `pnpm generate` from your Contrail config.
  Don't edit by hand.

All three are picked up by `lex-cli generate` (see `lex.config.js` `files` glob).
